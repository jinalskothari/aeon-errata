# Aeon's Surreal Renaissance — Errata & Enhancements

Errata, factual inaccuracies, broken examples, and enhancement suggestions collected while reading *[Aeon's Surreal Renaissance](https://surrealdb.com/learn/book)* — the official SurrealDB learning book by Dave MacLeod.

## Structure

Reports are filed in batches as chapters are read, one file per chapter range:

```
errata.md              ← master index (start here)
errata-overall.md      ← book-wide issues not tied to a specific chapter
errata-ch04-ch06.md    ← issues found in Chapters 4–6
errata-chXX-chYY.md    ← future batches follow the same pattern
```

Book-wide issues (e.g. Surrealist block scoping, UI bugs) go into `errata-overall.md` and are updated continuously as new ones are found, regardless of which chapter prompted them.

**[→ View the full index](errata.md)**

## Categories

Each issue is tagged with a category:

| Category | Meaning |
|----------|---------|
| **Error** | Something is outright wrong or broken |
| **Factual Inaccuracy** | The book describes behaviour that does not match reality |
| **Example not working** | A code example fails to run or produce the shown output |
| **Enhancement** | A minor improvement that would help clarity or correctness |
| **Suggestion** | An open question or idea worth considering |

## Impact levels

| Impact | Meaning |
|--------|---------|
| **Maximum** | Blocks the reader from completing the chapter |
| **Significant** | Causes meaningful confusion or incorrect mental models |
| **Medium** | Noticeable but workable around |
| **Minimal** | Cosmetic or very minor |
